If you need to delegate method to a custom class (not AR) and pass object as parameter to this class, you can use next code:
{% highlight javascript %} class One < ActiveRecord::Base # ... delegate :some_method, to: :name_for_the_class def name_for_the_class @delegator ||= ::ClassToDelegate.new(self) end end {% endhighlight %}
