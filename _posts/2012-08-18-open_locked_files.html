При настойке базы данных Postgresql возникла острая необходимость отредактировать файл  /etc/postgresql/9.1/main/pg_hba.conf

Но все попытки заканчивались неудачей - владелец файла Postgres и я открыть его не могу. Тогда я подумал, что было бы неплохо сделать запуск любых файлов от имени администратора - как в Windows. И к моей радости оказалось, что такое решение уже есть. Итак публикую суть:

Выполняете нижеприведенные команды и у вас в стандартном файловом менеджере Ubuntu 12.04 (Unity) - Nautils'е появляется пункт "Открыть как администратор". Открываем терминал, поехали:

{% highlight javascript %} sudo add-apt-repository ppa:nae-team/ppa sudo apt-get update sudo apt-get install nautilus-actions nautilus-actions-extra {% endhighlight %}
