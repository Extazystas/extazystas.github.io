Иногда, во время работы, нам интересен исходный код гема, нужно посмотреть на внутреннюю логику.

Для этого выполним команду (на примере гема devise):

{% highlight javascript %} bundle open devise {% endhighlight %}

Если в консоли появилось сообщение о том, что нужно задать переменную EDITOR и ничего не открылось, делаем следующее:

{% highlight javascript %} sublime ~/.bashrc {% endhighlight %}

В открывшемся файле добавляем эту переменную и имя редактора:

{% highlight javascript %} export EDITOR=sublime {% endhighlight %}

Profit!
