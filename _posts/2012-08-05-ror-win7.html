Итак, Вам нужно установить Ruby On Rails под Windows 7?

Отлично, для этого существует замечательный кит Railsinstaller, которые помогает избежать траты времени и сил при установке SQL, Git и т.д....
Установили, ввели в консоль своё имя и e-mail - появляется куча ошибок в консоли, но RoR Вроде как установлен.

Проверяем:

{% highlight javascript %} rails new ProjectName {% endhighlight %}

Создается проект (набор папочек и файлов) - обычно в C:/Sites
Пробуем зайти на http://localhost:3000 и... страница не найдена.
пишем в консоли

{% highlight javascript %} cd Project name {% endhighlight %}

Затем оказываемся в папке C:/Sites/Projectname
пишем:

{% highlight javascript %} rails server {% endhighlight %}

Консоль выводит нам:

{% highlight javascript %} ←[31mCould not find gem 'jquery-rails (= 2.0.0) x86-mingw32' in any of the gem sources listed in your Gemfile.←[0m ←[33mRun bundle install to install missing gems.←[0m {% endhighlight %}

Скачиваем гемы bundle и jquery-rails с сайта http://rubygems.org/
Дальше самое важное!
1) Закидываем скачанные файлы в папку C:/Users/USERNAME - где USERNAME - имя вашего профиля в Windows.
2) Пишем в консоли SET HOME=%USERPROFILE% (где USERPROFILE = USERNAME = имя вашего профиля в системе - на кириллице не проверял (лучше используйте латиницу для имени своего профиля).
3) пишем в консоли:

{% highlight javascript %} gem install bundle После его установки пишем: gem install jquery-rails {% endhighlight %}

4) Все! После установки гема выполняем команду rails server из папки нашего проекта и заходим на Localhost:3000
