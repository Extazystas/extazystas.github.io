In your terminal, enter:

{% highlight javascript %} sudo gedit /usr/share/applications/sublime.desktop {% endhighlight %}

In open file you should enter next content:

{% highlight javascript %} [Desktop Entry] Version=1.0 Name=Sublime Text 3 # Only KDE 4 seems to use GenericName, so we reuse the KDE strings. # From Ubuntu's language-pack-kde-XX-base packages, version 9.04-20090413. GenericName=Text Editor Exec=sublime Terminal=false Icon=/usr/lib/Sublime Text 3/Icon/48x48/sublime_text.png Type=Application Categories=TextEditor;IDE;Development X-Ayatana-Desktop-Shortcuts=NewWindow [NewWindow Shortcut Group] Name=New Window Exec=sublime -n TargetEnvironment=Unity {% endhighlight %}

If it's not working correct, replace 'sublime' with you shortcut for Sublime text (for my Sublime text 3 I have 'subl', instead 'sublime')
Hopefully now Sublime Text 2 or 3 will work fine from your right click context menu.
