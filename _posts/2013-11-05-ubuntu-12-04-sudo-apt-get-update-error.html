Recently, I have to update my ubuntu packages with a old good-known command 'sudo apt-get update' and I was fucked up

I got message like this:

Solution is very simple: The medibuntu repository has closed. You need to remove references to medibuntu from your sources.list file. If you don't want to edit the sources.list file itself, you can disable medibuntu in update manager

Just open your update magaer ($ update-manager) and remove medibuntu repo in settings)

Profit!
